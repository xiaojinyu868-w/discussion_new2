## 💡 我们解决了什么问题？

传统会议工具只是"旁观者"——被动记录、事后整理。但会议真正需要的是一个**数字参会者**：

- 🎧 **全程在场**：不是事后转写，而是实时理解每一句话
- 🧠 **深度理解**：不只是文字，而是洞察潜台词、捕捉情绪、识别分歧
- 💡 **主动洞察**：不等你问，主动提醒跑题、推送灵感、生成共识图

**MeetMind** 是您会议中最懂行的那位——它真正"参与"会议，而不只是"记录"会议。

---

## 🏗️ 四层认知对齐模型

| 层级 | 名称 | 能力 | 价值 |
|------|------|------|------|
| L1 | **信息层** | 全息记录 | 毫秒级语音转写，一字不漏 |
| L2 | **认知层** | 认知同频 | 潜台词解读、情绪识别、分歧发现 |
| L3 | **视觉层** | 思维显影 | 抽象讨论 → 图表/海报/创意图 |
| L4 | **战略层** | 决策支撑 | 主动推送洞察，辅助决策 |

## 🧩 核心功能

### 认知技能（人主动触发）

| 技能 | 说明 |
|------|------|
| 🔮 **认知转译** | 解读潜台词，揭示"话外之音" |
| ⚡ **破局灵感** | 基于讨论内容，激发创意思路 |
| 🎯 **议程守护** | 检测跑题，守护会议聚焦 |

### 思维显影（视觉化生成）

| 类型 | 说明 |
|------|------|
| 📊 **数据图表** | 口头数据 → 雷达图/柱状图/流程图 |
| 🎨 **创意图像** | 概念讨论 → 示意图/创意配图 |
| 📋 **逻辑海报** | 结构化信息 → 一图胜千言 |

### Agent 智能联动（AI 主动推送）

数字参会者每 **10 秒**自动巡检，检测到以下情况时**主动介入**：

| 检测类型 | 触发条件 | 自动动作 |
|----------|----------|----------|
| 📈 数据洞察 | 检测到数字/百分比/趋势 | 自动生成图表 |
| ✅ 决策识别 | 检测到"确定/同意"等决策词 | 生成决策卡片 |
| 🎯 跑题提醒 | 语义偏离主议题 | 自动聚焦提醒 |
| 🔄 冗余警告 | 重复讨论相同内容 | 生成精简建议 |

### 上下文问答（记忆盒子）

随时提问，数字参会者基于全量上下文回答：

- "刚才谁提到了预算？"
- "前 10 分钟的核心结论是什么？"
- "总结一下小明的所有发言"

## 🚀 快速开始

### 环境要求
- Node.js ≥ 18
- 阿里云通义听悟账号
- Qwen API Key
- Nano Banana Pro API Key

### 启动后端

```bash
cd backend
npm install

# 创建 .env 文件配置 API Keys
npm run start:dev
```

### 体验演示

**直接打开 `demo_show/index.html`** 即可体验完整功能。
