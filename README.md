# MeetMind · 视界 — 认知对齐工具

> **不只是记录，而是真正理解** — 帮你与老师、与他人保持认知同步

---

## 💡 我们解决什么问题？

传统工具解决的是「记不住」，我们解决的是「听不懂」。

会议和课堂中最大的信息损耗不是「没听到」，而是**没能与对方的认知对齐**：
- 学生跟不上老师思路，漏掉言外之意
- 参会者听不懂潜台词，错过关键信号

**MeetMind** 是一个**认知对齐工具**——帮你消除理解上的偏差，真正听懂对方想说什么。

---

## 🎯 两大对齐场景

| 场景 | 对齐目标 | 你的痛点 | AI 帮你 |
|------|---------|---------|--------|
| **🎓 课堂模式** | 老师 | 跟不上思路，漏掉言外之意 | 像学霸同桌，实时翻译老师的深层意图 |
| **💼 会议模式** | 其他参会者 | 听不懂潜台词，错过关键信号 | 像读心高手，解读每个人话语背后的想法 |

---

## 🧠 三层对齐能力

### 1. 被动对齐
AI 持续监听，发现你可能「没听懂」时，主动轻推提醒。

### 2. 主动对齐（认知技能）
感到困惑时，一键触发特定技能：

| 场景 | 技能 | 作用 |
|------|------|------|
| 课堂 | **对齐老师思维** | 捕捉老师话语中的认知密度最高点，翻译教学意图 |
| 课堂 | **拓展认知边界** | 建立知识连接，与更广阔的知识体系对齐 |
| 课堂 | **对齐学习目标** | 梳理核心知识点，确保与课程目标保持同步 |
| 会议 | **对齐他人意图** | 解码话语背后的真实诉求，识别潜台词和立场 |
| 会议 | **破局新视角** | 跳出思维定式，与更高维度的解决方案对齐 |
| 会议 | **对齐会议目标** | 守护会议主线，确保团队与目标保持同步 |

### 3. 深度对齐（智能问答）
随时提问，AI 基于全量上下文帮你「补课」：
- "老师刚才那句话是什么意思？"
- "张三这么说是在暗示什么？"
- "我漏听了什么重要的点？"

## 🚀 快速开始

### 环境要求
- Node.js ≥ 18
- 阿里云 DashScope API Key（用于语音转写和 LLM）

### 启动后端

```bash
cd backend
npm install

# 创建 .env 文件配置 API Keys
# DASHSCOPE_API_KEY=your_api_key
npm run start:dev
```

### 启动前端

```bash
cd meetmind-frontend
npm install
npm run dev
```

      ├── llm/              # LLM 适配器

      └── quota/            # 配额管理
